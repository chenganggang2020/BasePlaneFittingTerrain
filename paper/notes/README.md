# Notes Folder

Use this folder for lightweight manuscript support material that should stay separate from code and experiment results.

- `extracted_text/`
  PDF text extraction, OCR exports, and other raw reading inputs.
- `method_gap/`
  Notes that compare the paper method with the current implementation.
- `reading_notes/`
  Reading summaries, key formulas, and interpretation notes.
- `draft_outline/`
  Writing outlines, figure mapping, and chapter planning notes.

Recommended habit:

- Put raw imported text in `extracted_text/`.
- Put stable analysis notes in `method_gap/` or `reading_notes/`.
- Put manuscript planning material in `draft_outline/`.
